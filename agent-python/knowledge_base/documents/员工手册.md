# 员工手册

## 第一章 总则

### 1.1 公司简介
本公司是一家专注于人工智能技术研发的高科技企业，致力于为客户提供优质的AI解决方案。

### 1.2 员工行为准则
- 遵守国家法律法规和公司规章制度
- 诚实守信，保守公司机密
- 尊重同事，团结协作
- 积极进取，追求卓越

## 第二章 入职与离职

### 2.1 入职流程
1. 提交入职材料（身份证、学历证明、离职证明等）
2. 参加新员工培训
3. 签署劳动合同
4. 开通公司账号和邮箱

### 2.2 试用期规定
- 新员工试用期为1-3个月
- 试用期内工资为正式工资的80%
- 试用期考核合格后转为正式员工

### 2.3 离职流程
1. 提前提交离职申请
2. 完成工作交接
3. 办理离职手续
4. 结算工资和福利

## 第三章 考勤制度

### 3.1 工作时间
- 周一至周五：9:00-18:00
- 午休时间：12:00-13:30
- 周六、周日休息

### 3.2 打卡规定
- 每日上下班需打卡
- 迟到15分钟内不扣分
- 迟到超过15分钟按事假处理

## 第四章 假期制度

### 4.1 年假
- 入职满1年可享受5天年假
- 入职满3年可享受10天年假
- 入职满5年可享受15天年假

### 4.2 病假
- 需提供医院证明
- 病假期间工资按基本工资的70%发放

### 4.3 事假
- 需提前申请
- 事假期间无工资

## 第五章 薪酬福利

### 5.1 工资结构
- 基本工资
- 绩效工资
- 补贴（交通、餐补等）
- 年终奖金

### 5.2 社保公积金
- 公司按国家规定缴纳五险一金
- 个人承担部分从工资中扣除

## 第六章 员工发展

### 6.1 培训机会
- 新员工入职培训
- 专业技能培训
- 管理能力培训

### 6.2 晋升机制
- 定期绩效考核
- 内部晋升优先
- 公平公正的晋升机会
